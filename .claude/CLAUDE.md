This file provides guidance to Claude Code (claude.ai/code) when working with code in all repositories.
The following are personal preferences for all projects, this include code styling preferences, personal tooling
architectures, etc.

## Plugins

`~/.zshrc` has some powerful snippets that should be used when working with code. These include:

- OMZP::git: Provides git aliases and functions for easier git management. Always use `g` instead of `git` when running git commands in the shell.
- OMZP::docker: Provides docker aliases and functions for easier docker management.
- OMZP::docker-compose: Provides docker-compose aliases and functions for easier docker-compose management.
